# WikiSheets

Make sure to download the required `spaCy` module by running `python -m spacy download en_core_web_sm` before running any of the included scripts.

## project.yaml

The `project.yaml` should be set up as follows:

```
entrez:
  entrez_email: 
wikibase:
  full_bot_name: 
  bot_name: 
  bot_password: 
  mediawiki_api_url: 
  sparql_endpoint_url: 
  wikibase_url: 
  wikibase_name: 
```
